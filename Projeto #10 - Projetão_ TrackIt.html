Projeto #10 - Projetão: TrackIt
Descrição
Neste projeto você irá construir uma aplicação completa de acompanhamento de hábitos! Com direito a cadastro, login e muitas bibliotecas
Importante:
Lembre-se que para aprender algo novo afim de utilizá-lo em algum projeto, podemos antes de aplicar de fato o conhecimento no projeto (o quê pode ser muito complexo), criar um POC (Proof of Concept/Prova de Conceito).
Um POC pode ser um projeto ou snippet simples que serve apenas para aplicar o conceito/tecnologia na prática e já sentir as dores de utilizá-lo, para não misturá-las com as dores do seu projeto. Não precisa entrar a fundo nos detalhes para criar um POC, a
Depois de sofrer bastante para entender o conceito/tecnologia na construção do POC, você pode partir para a utilização do mesmo no seu projeto sem remorso, olhando a documentação a fundo para não ter nenhuma ponta solta.

Requisitos
Geral
Manipule o HTML usando somente React (você não deve manipular o DOM diretamente com querySelector, innerHTML, classList)
Para controlar os dados dinâmicos da aplicação, utilize as ferramentas de gerenciamento de estado do React (não utilize variáveis globais)
Para estados globais (como usuário logado e progresso do dia) utilize ContextAPI. Local Storage só deve ser usada para armazenar as credenciais do usuário, enviadas pelo servidor. Session Storage não deve ser utilizado.
Versionamento usando Git é obrigatório, crie um repositório público no seu perfil do GitHub
